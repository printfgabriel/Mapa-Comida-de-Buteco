# Mapa Comida de Buteco

Instruções rápidas para rodar o projeto em Linux.

Clone o repositório:

```
git clone git@github.com:printfgabriel/Mapa-Comida-de-Buteco.git
cd Mapa-Comida-de-Buteco
```

Crie e ative o ambiente virtual:

```
python3 -m venv venv
source venv/bin/activate
```

Instale as dependências:

```
pip install -r requirements.txt
```

Rode a aplicação:

```
python3 main.py
```

Depois de iniciar, abra no navegador o endereço mostrado no terminal. Normalmente será:

```text
http://127.0.0.1:8050/
```

## Observações

Na interface, depois de digitar o valor do raio ou a diagonal do retângulo, pressione `Enter` para aplicar a busca.
