# Mapa Comida de Buteco

Instruções rápidas para rodar o projeto em Linux ou Windows.

Clone o repositório:

```
git clone git@github.com:printfgabriel/Mapa-Comida-de-Buteco.git
cd Mapa-Comida-de-Buteco
```

Crie o ambiente virtual:

Linux:

```
python3 -m venv venv
```

Windows:

```
python -m venv venv
```

Ative o ambiente virtual:

Linux:

```
source venv/bin/activate
```

Windows:

```
venv\Scripts\activate
```

Instale as dependências:

```
pip install -r requirements.txt
```

Rode a aplicação:

Linux:

```
python3 main.py
```

Windows:

```
python main.py
```

Depois de iniciar, abra no navegador o endereço mostrado no terminal. Normalmente será:

```
http://127.0.0.1:8050/
```

## Observação

Na interface, depois de digitar o valor do raio ou a diagonal do retângulo, pressione `Enter` para aplicar a busca.
